OSU Navigation Bar Example: Link to search

This alternate version of the navigation bar has a hyperlink to osu.edu's search.

A red/gray common navigation bar at the top of the page is the recommended primary common element among all Ohio State service sites and department/college web pages. It's a minimal primary element that reasonably attempts to tie together thousands of web pages under decentralized control. To maintain consistent usage across the university, please do not modify the navigation bar beyond the text inside <p id="site-name">. In some cases the "osu-Navbar" background color (#EAEBEB) may be changed for better design consistency with an existing site's background. For more information see: http://osu.edu/resources/navbar.html. If in doubt contact University Relations-New Media.
