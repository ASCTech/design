Ohio State Navigation Bar Template

Revised July 2008

Provided with this "Read me" file are ready-to-use Ohio State Navigation Bar html and approved Ohio State graphics for use by Ohio State web developers. For more information regarding appropriate use and implementation of these graphics and templates, please consult the Ohio State Identity Guidelines or contact University Marketing Communications, New Media. For more information about Accessible design and universal web design help to accommodate students with learning and/or physical disabilities, visit the Web Accessibility Center

A red/gray common navigation bar at the top of the page is the required primary common element among all Ohio State service sites and department/college web pages. It's a minimal primary element that reasonably attempts to tie together thousands of web pages under decentralized control. To maintain consistent usage across the university, please do not modify the navigation bar or any of the text within it.
